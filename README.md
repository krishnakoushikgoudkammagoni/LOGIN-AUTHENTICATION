🔐 Login Authentication System

A simple Login Authentication System built using HTML, CSS, and JavaScript. This project provides a basic login interface where users enter a username and password, and the system verifies the credentials using JavaScript.

🚀 Features

👤 User login interface

🔑 Password authentication

⚡ Instant login validation using JavaScript

🎨 Clean and simple user interface

🛠️ Technologies Used

HTML – Structure of the login page

CSS – Styling and layout design

JavaScript – Login authentication logic

📂 Project Structure
login-authentication-system/
│
├── index.html      # Login page
├── style.css       # Styling for login UI
├── script.js       # Authentication logic
└── README.md       # Project documentation
⚙️ How to Run the Project

Clone the repository

git clone Link :- https://krishnakoushikgoudkammagoni.github.io/LOGIN-AUTHENTICATION
